知識點

問答

  1. 請問一下,暴力破解法的方式是什麼?(就是一個一個將可能的數字去比對)
  2. 使用什麼指令可以讓它可以跑無窮迴圈呢?(//noprotect)
  3. 請問一下什麼是排除破解法?(就是將不相同的幾A幾B的數字移除掉)

常見問題

  1. 使用暴力破解法會沒有破解完成
  2. 原來的程式碼是 : 
            var list = getList();
        //noprotect
        for (var i = 0; i < list.length; i++) {
            var g = list.pop();
            var result = guess(g);
            if (result.a == 4) {
                break;
            }
            for (var j = 0; j < list.length; j++) {
                var result2 = diff(list[j], g);
                if (result.a != result2.a || result.b != result2.b) {
                    list.splice(j, 1);
                }
            }
        }
    後來改成這樣 
            function crack () {
            var list = getList();
            //noprotect
            for (var i = 0; i < 5040; i++) {
                var g = list.pop();
                var result = guess(g);
                if (result.a == 4) {
                    break;
                }
                for (var j = 0; j < list.length; j++) {
                    var result2 = diff(list[j], g);
                    if (result.a != result2.a || result.b != result2.b) {
                    list.splice(j, 1);
                    j--;
                    }
                }
            }
        }
