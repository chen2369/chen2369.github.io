知識點

問答

  1. 這一段程式碼在做什麼?(rand)
    ANS:
    透過random.random()隨機產生一個0~1之間的小數
    +1氣球產生的機率是40%、+5氣球產生的機率是20%、+10氣球產生的機率是10%、*2氣球產生的機率是10%、/2氣球產生的機率是10%、炸彈氣球產生的機率是5%、加時間氣球產生的機率是5%、
  2. random.randrange 和 random.random() 的差別是什麼?
    ANS:
    random.randrange是在兩個數字之間隨機產生一個整數
    random.random()隨機產生一個0~1之間的小數
