知識點

問答

  1. 如何建立不同的水晶和泡泡並且儲存？Ans: 利用限定執行次數的迴圈，與串列的 append() 方法。
  2. 如何做出水晶高低起伏的效果？Ans: 利用前一個水晶的 Y 座標（init_y）再加上一隨機數，再利用 300~600 的限制，避免太大的高低落差。

常見問題

  1. 對於 Python 中 for-loop 用法不理解（對於 range() 不熟）：直接利用實驗示範。
  2. 免除 init_y 的寫法：可以直接利用 rocks[i-1] 來存取前一個水晶物件

延伸挑戰

  1. 隨著 level 增加，加快移動速度。
  2. 加入生命值的機制，讓玩家有更多次失敗的機會（碰到水晶後，回到通道中央）。
  3. （進階）嘗試加入第二個角色，並設計勝負機制，也加入前進後退功能。
  4. （進階）嘗試寫出讓潛水艇自己修正航線的程式。
