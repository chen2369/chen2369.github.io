常見問題

  1. 學生易錯、易問： 這課還沒開始使用資料庫！！
    而是專門在複習『物件』與『陣列』等知識點，因為之後會一直用到 <td><div class='icon icon-${pokemons[i].index}'></div></td>
    icon部分只要class加上對應的 icon-${pokemons[i].index} 就能出現圖片
    這部分在後台老師寫了一串指令，學生可以不用了解這部分如何實作，只要知道對應到就能出現，而底層的做法方向是
    class：icon 是用來切 40px 40px，剛好是大圖中一個角色的大小
    class：icon-id 是用來決定應該定位到大圖的哪個座標
  2. 問：為什麼剛剛儲存的資料不見了
    答：因為這堂課我們將資料並沒有存在資料庫中，從新整理後陣列就會重新初始化了
  3. 如何取得pokemons陣列中第一筆物件的name屬性？
    pokemons[0].name
  4. 多筆資料可以使用什麼來儲存?它可以使用哪四個方法在前後塞與吐值?
    陣列
    前：shift()、unshift() 後：pop()、push()
  5. 如何創造一個物件people，包含屬性name='小明'，方法say 能alert出自己名字
    people = { name:'小明', say:function(){ alert(this.name) } }
  6. 如何使用『字串模板』?變數在裡面需要用什麼包起來?
    使用 ` ` 符號
    ${}
  7. 透過 jquery 什麼方法將 tbody清空
    $('tbody').empty()
  8. 透過 jquery 什麼方法將資料印上tbody
    $('tbody').append(row)
