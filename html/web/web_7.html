常見問題

  1. 學生斜線容易用錯，要使用反斜線
  2. 畫面在載入時會先顯示要被隱藏的圖片，接著才會被jq指令隱藏，可以當作挑戰題讓學生暸解網頁的生命週期，並思考看看如何解決這個狀況
  3. 解法：針對id元件使用CSS的隱藏方式
  4. 延伸挑戰：判斷空白(trim)以及去除重複(indexOf)的選項 隱藏圖片的三種做法
    影片中介紹的利用jq 的hide方法(缺點：會有執行的時間差，會先顯示圖片再隱藏)
    利用CSS設定選擇器{display : none}，可以在一開始載入就隱藏圖片
    HTML只放置一個圖片標籤，利用JQ去替換src的內容$(選擇器).attr(‘src’,’圖片名稱’)，缺點：難度比較高
