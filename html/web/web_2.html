  7. css diner的第七關，學生會出現css空格和點號的問題
  8. 例如有一個html標籤，是<div class="one two">，此時是指這個div同時有兩個選擇器，one和two，並不是一個選擇器叫onetwo。
  9. 因此對應到css時，會出現三種常見問題，這三種css選擇器上的編寫方式差異如下：
    第一個的one和two中沒有包含空格，這個的意思表示，某個區塊必須同時具有one和two的class時，才能被CSS 所選擇到。
    第二個的one和two中包含空格，意思是指，我必須要是在one裡面的two，才會被選擇到。
    第三個的one和two中包含逗號，意思是指class中有one或two，都會被編輯器所選擇到。
