常見問題

  1. </html>與</body>標籤因為離前頭太遠，有時常被學生刪掉。
  2. 標籤裡的單雙引號雖然是皆可辨識，但如果前面單引號、後面雙引號就是錯的，例如<img src="">，建議統一都用同一種引號。
  3. <a href=" ">、<script src=" ">、<img src=" ">，學生會混淆。
  4. 影片的單引號跟雙引號的問題，在教學影片中會出現兩種，但html5建議使用雙引號，而js使用單引號。
  5. 教室鎖YT，以至於在嵌入影片的章節無法操作，是否提供部分影片至學苑或提供其他雲端連結。
