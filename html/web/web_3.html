  1. 一到三個等於＝，學生會搞混。
  2. 統整一下等於＝的用法：
  3. 1個等於＝：用在宣告或運算。例如var a=1或a=b。
  4. 2個等於＝：用在判斷式（只看值、不看類別）。例如1=＝”1”或1=＝true。
  5. 3個等於＝：用在判斷式（看值也看類別）。例如1===1是對的，但1===”1”是錯的。
