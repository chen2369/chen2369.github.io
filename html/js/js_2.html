常見問題

  1. 一開始認識變數的課程影片中是寫在主控台，若學生找不到愣住請提醒一下學生，寫在左邊也是可以的。
  2. 執行後變數顯示會疊在一起，影片有提過了，但有些學生做的時候還是會忘記。
  3. 真假值首字為小寫，與python不同；如果學生找不到或（|）的符號要跟他說一下。
  4. 另可以再考考學生一個與兩個等號同時出現在一行程式碼中時會發生什麼事，確定學生有理解影片內容，也可以結合布林值運算與程式逐行執行的性質出一些難一點的題目考考學生，例如：
    5. var a = true
    var b = false
    a = a && b
    b = a || b
  5. 程式小測驗章節請學生做完後停留在該畫面舉手，老師才能去查看分數並且了解學生的學習狀況。
  6. 測驗完如果還有時間的話可以再進行回家作業的練習，不過除非以前就學過不然學生通常進度沒有那麼快，因此若有時間但不夠實作的話，可以用口頭問答的方式詢問學生會怎麼做這兩個作業（梯形面積＆溫度轉換）。
  7. 小測驗做完之後請他停留在送出的畫面，才可以看到答題狀況
  8. 這一課比較短，要準備額外練習
