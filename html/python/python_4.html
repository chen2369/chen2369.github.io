問答

常見問題

  1. 射箭時碰到蘋果學生會誤以為真正的把箭插在蘋果上，需要跟學生解釋是把蘋果隱藏，把箭的造型換成有插著蘋果的造型。
  2. 告知學生會把apple hidden來，並且把arrow的造改成有插上apple的。
  3. 當箭超出場景需要把箭設定回原位，以及把弓的角度回條回來。學生會忘記把弓角度射回原來的角度。
  4. 學生容易打錯字。

延伸挑戰

  1. 可以讓學生嘗試「即使射到了蘋果還是扣一支箭」。
