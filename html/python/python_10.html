知識點

問答

  1. 串列是什麼？我們在哪裡有用到串列？為什麼要用串列儲存？Ans: 關卡 X, Y 座標。
  2. 我們針對燃料設計了哪些機制？Ans: 初始值、消耗時機、動畫。
  3. （進階）稍微解釋火焰怎麼產生的？

常見問題

  1. 有部份學生仍會對 vx. vy 不理解：主要以實驗說明
  2. 對於串列相關的知識點不理解（索引值從 0 開始算、取值）：以實驗引導說明。
  3. 碰到遊戲牆壁時，info 角色無法順利出現：於【K.等待開始】章節中有提及解法。
  4. 遊戲開始直接 Game Over：注意 levels.costume_id, goal_x, goal_y, target 等變數設定時是否有誤。
  5. 原本是讓 target 座標持續定位，可以改寫在碰到 target 的條件判斷內。

延伸挑戰

  1. 加入更多按鍵，使玩家更好操作叩叮。
  2. 加入生命值的機制，讓玩家有更多次失敗的機會（碰到牆壁後，回到關卡出發點）。
  3. （進階）嘗試加入第二個角色，並設計碰撞機制。
