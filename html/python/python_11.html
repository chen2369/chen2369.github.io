知識點

問答

  1. 蛇移動方向的限制是什麼？為什麼要用到 next_direction 這個變數？
  2. 為何未檢查 bodies 的長度，而直接使用 bodies.pop() 會出錯？
  3. 解釋 bodies.pop()、[]+bodies 和 bodies[-1] 的意思。
  4. 蛇實際上是怎麼移動的？Ans: 取出陣列最末值（pop()），放到頭的位置（move_to(head)），再從最前端加入陣列後（[]+bodies），判斷並更新造型。

常見問題

  1. 此課將畫面分較多格（一格 60 點）進行位置推算，部份學生容易混亂：可提供紙筆紀錄與計算，並且直接實驗給學生看。
  2. 做完【H. 移動身體】章節後，蛇不會動：有可能是條件判斷式的階層放錯。
  3. 蛇行動過程中外觀異常（斷掉等狀況發生）：可以檢查移動頭和換造型函式的順序，要移動完成後才可以換造型。
  4. 部份影片的內容不連貫（換造型的函式 update_bodies_costume()）：直接向學生解釋有做過難易度調整。
  5. 遊戲完成後背景圖片可以改成 bg.jpg（預設為 bg_grid.jpg），網格會消失
