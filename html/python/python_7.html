常見問題

  1. 本課程教學影片解說很詳細，學生較少有問題。不過在反彈角度的部分，學生比較容易只知道結論，但對原理則不太了解。老師可以在這方面跟學生確認。解說則可以以Scratch的腳色方向的例子作為輔助講解，右半圈0~180度、左半圈0~ -180，而非360度一圈。
  2. 可以出挑戰題，如何讓一邊的擋板交由電腦控制?
  3. 可以讓擋板追隨球的y軸。
