知識點

問答

  1. 如何讀取輸入筐的值,並新增到task資料表中
    var t = $('#input').val()
    task.create({text:t})
  2. 如何移除全部task資料表中的資料
    task.delete({})
  3. 為什麼要使用『動態事件綁定』來觸發按鈕
    因為表格中的內容都是透過js去另外印上畫面的，並不是一開始ＨＴＭＬ就有，若用$('btn').click(function(){})將無法正常的觸發
  4. 本週為了移除清單中的事項，動態事件綁定的語法是怎麼寫的呢?
    $('#list').on('click','.delete,function(){})
  5. 以上一題為例，為什麼動態事件綁定我們綁定在#list而不是直接綁定.delete呢？
    因為#list不是動態產生的元素，而.delete是動態產生的
  6. 說明為什麼不能使用文字內容來找尋要移除的資料，而是要使用ＩＤ？
    因為文字內容可能會有重複的，而ＩＤ不會
  7. 說明如何移除被點擊的指定的項目?
    在div中加入data-id屬性，在按鈕被點擊時取得這個值進行移除
  8. 上一題的程式應該怎麼寫？
    var id = $(this).parent().data('id')
    task.delete({id:id})

常見問題

  1. Ｑ：為什麼這邊需要改為 .on(‘click’,) 寫法
